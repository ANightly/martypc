
Marty requires the following ROMs be placed in this directory.

IBM 5150 
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

One of the following main system BIOS files:

    BIOS_IBM5150_27OCT82_1501476_U33.BIN **** preferred ****
    8,192 bytes
    md5sum: f453eb2df6daf21ec644d33663d85434

-- or --

    BIOS_IBM5150_19OCT81_5700671_U33.BIN
    8,192 bytes
    md5sum: 6a1ed4e3f500d785a01ff4d3e000d79c

-- or --

    BIOS_IBM5150_24APR81_5700051_U33.BIN
    8,192 bytes
    md5sum: 6338a9808445de12109a2389b71ee2eb

Optionally, also include one of the BASIC ROMS:

    basic_v1.rom
    32,768 bytes
    md5sum: 2ad31da203a49b504fad3a34af0c719f

-- or --

    basic_v101.rom
    32,768 bytes
    md5sum: eb28f0e8d3f641f2b58a3677b3b998cc

IBM/Xebec 20MB Fixed Disk Adapter
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

If you wish to add a hard disk to the IBM XT 5160 machine, you MUST supply
a hard disk controller BIOS:

    IBM_XEBEC_62X0822_1985.BIN
    4,096 bytes
    md5sum: 66631d1a095d8d0d54cc917fbdece684
